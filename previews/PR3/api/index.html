API overview

APISummary
NonreentrantBackoffSpinLockA non-reentrant exponential backoff spin lock.
NonreentrantCLHLockA (non-reentrant) CLH "spinnable" lock that provides first-come-first-served fairness. Keyword argument nspins::Integer can be passed to acquire to specify a number of spins tried before falling back to "cooperative" waiting in the Julia scheduler.
ReentrantBackoffSpinLockA reentrant exponential backoff spin lock.
ReentrantCLHLockA reentrant CLH "spinnable" lock that provides first-come-first-served fairness. Keyword argument nspins::Integer can be passed to acquire to specify a number of spins tried before falling back to "cooperative" waiting in the Julia scheduler.
TaskObliviousLockA lock that can be released in a task that did not acquire the lock. It does not support reentrancy.
acquire(lock)Acquire a lock. It is equivalent to Base.lock(lock) but it may support additional keyword arguments.
acquire_then(f, lock; acquire_options...) -> yExecute a thunk f in a critical section protected by lock and return the value y returned from f. Keyword arguments are passed to acquire.
race_acquire(lock) -> isacquired::BoolTry to acquire lock and return true on success and false on failure.
release(lock)Release a lock. It is equivalent to Base.unlock(lock).
try_race_acquire(lock) -> Ok(nothing) or Err(reason)Try to acquire lock and return Ok(nothing) on success. Return an Err wrapping a value explaining a reason of failure.
