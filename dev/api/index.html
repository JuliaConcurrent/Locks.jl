API overview

APISummary
@once codeExecute code at most once within the lifetime of the Julia process. It evaluates to the result of the first evaluation whenever the same location of the code is evaluated later.
@tasklet codeCreate an object that is a memoized version of () -> code but also acts like a Promise that is not settable.
Guard(data)Guard mutable data. Use guarding to obtain exclusive access to data.
NonreentrantBackoffSpinLockA non-reentrant exponential backoff spin lock.
NonreentrantCLHLockA (non-reentrant) CLH "spinnable" lock that provides first-come-first-served fairness. Keyword argument nspins::Integer can be passed to acquire to specify a number of spins tried before falling back to "cooperative" waiting in the Julia scheduler.
Promise{T}()Create a promise of type T; i.e., a memory location that holds a value of type T that can be set once and retrieved asynchronously.
ReadWriteGuard(data)Guard mutable data. Use guarding and guarding_read to obtain exclusive ("write") and shared ("read") access to data.
ReentrantBackoffSpinLockA reentrant exponential backoff spin lock.
ReentrantCLHLockA reentrant CLH "spinnable" lock that provides first-come-first-served fairness. Keyword argument nspins::Integer can be passed to acquire to specify a number of spins tried before falling back to "cooperative" waiting in the Julia scheduler.
TaskObliviousLockA lock that can be released in a task that did not acquire the lock. It does not support reentrancy.
ThreadLocalStorage{T}(factory)Create a thread-local storage of type T created by factory().
acquire(lock)Acquire a lock. It is equivalent to Base.lock(lock) but it may support additional keyword arguments.
acquire_then(f, lock; acquire_options...) -> yExecute a thunk f in a critical section protected by lock and return the value y returned from f. Keyword arguments are passed to acquire.
guarding(f!, guard)Apply f! to the data wrapped in guard while obtaining exclusive access.
guarding_read(f, guard)Apply f to the data wrapped in guard while obtaining shared access.
race_acquire(lock) -> isacquired::BoolTry to acquire lock and return true on success and false on failure.
race_fetch_or!(thunk, promise::Promise{T}) -> value::TFetch an existing value or set value = thunk(). The thunk is called at most once for each instance of promise.
read_write_lock() -> (rlock, wlock)Return the read handle rlock and the write handle wlock of a read-write lock.
release(lock)Release a lock. It is equivalent to Base.unlock(lock).
spinloop()A hint to the compiler, runtime, and hardware that spinloop() is in the middle of a spin loop. Call this in a spin loop that requires some other worker threads to make forward progress in order for the current thread to make forward progress.
try_race_acquire(lock) -> Ok(nothing) or Err(reason)Try to acquire lock and return Ok(nothing) on success. Return an Err wrapping a value explaining a reason of failure.
try_race_fetch(promiselike) -> Ok(value::T) or Err(NotSetError())Try to retrieve a value if it is already set. Return Ok(value) on success and Err(NotSetError()) on failure.
try_race_fetch_or!(thunk, promise::Promise{T}) -> Ok(existing::T) or Err(computed::T)Fetch an existing value or set a computed value (computed = thunk()). The thunk is called at most once for each instance of promise.
try_race_put!(promise::Promise{T}, value) -> Ok(value′::T) or Err(OccupiedError(existing::T))Try to set a value in the promise.
